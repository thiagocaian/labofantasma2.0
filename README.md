LaboFantasma 2.0 is my personal offensive security laboratory — a living portfolio documenting my journey from CompTIA A+ to professional Penetration Tester in Australia. Every project, writeup, and script here represents real, hands-on practice aligned with industry standards and Australian cybersecurity hiring requirements.

This repository serves as both a **learning journal** and a **technical portfolio** for cybersecurity employers.
